Rules:
1. block is applied by default
2. you can define width and height. But default width always occupies the entire line
3. content always start in a new line (top-to-bottom)
4. no new content can appear besides it
5. content can overflow
6. whitespaces in content causes it to wrap below, unless there is enough space in the box to accomodate the content

Usage Requirements:
1. the flex display attribute requires a parent and several child elements to work properly
2. you must add display:flex to the parent's css property ONLY
3. the child elements do not need the flex property
4. Though only the parent has the flex property, flex behavior will apply to the child and not the parent.

Some extra notes about Flex and Block alignments

The box above has been given both:

2. align-items:center; //align vertically

hence it appears right in the middle of the flex-container 

 
Flex-Only Alignment Properties
justify-content and align-items (only works on flex elements)

//applies on the Main Axis of the flex-container (i.e. default: x-axis, horizontally)
justify-content: flex-start;
justify-content: flex-end;
justify-content: center;
justify-content: space-between;
justify-content: space-around;
justify-content: space-evenly;


//applies on the Cross Axis of the flex-container (i.e. default: y-axis, vertically)
align-items: flex-start;
align-items: flex-end;
align-items: center;
align-items: stretch;
align-items: baseline;

NOTE: if you are using flex-direction: column, the main axis will be set to y-axis instead of x-axis. hence, justify-content will apply vertically in this scenario.


Block-Only Alignment Properties
text-align and vertical-align (only works on block elements)

//always applies horizontally 
text-align: left;
text-align: right;
text-align: center;
text-align: justify;

//always applies vertically
vertical-align: baseline;
vertical-align: text-top;
vertical-align: middle;
vertical-align: bottom;
vertical-align: text-bottom;
vertical-align: super;
vertical-align: sub;

NOTE: direction of text-align and vertical-align cannot be switched
